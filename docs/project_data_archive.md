## What This Export Contains

The full teacher-student export is intentionally named and ordered:

- `01_gold_approved_annotations.csv` - human-approved production annotations used as the gold dataset.
- `02_gold_all_annotations_audit.csv` - all production annotations for audit and traceability.
- `03_gold_statistics/` - production statistics, taxonomy distributions, participant/reviewer CSVs, and infographic source files.
- `04_gold_silver_manifest.txt` and `04_gold_silver_manifest.json` - machine-readable and human-readable index of the export.
- `silver_batches/05_silver_unresolved_batch_XXX_of_YYY_<pool>.csv` - unresolved active source records, exported in numbered batches for automatic teacher labeling.

This archive is the main evidence source for the teacher-student stage: gold labels come from humans; silver labels are produced from unresolved records by automatic teacher and must be tracked separately.

  weak axes;
- simple anchor overrides overfit validation and did not improve test;
- the best full taxonomy configuration remains the cascade plus optional
  `responsible_party` pseudo-gold specialist at about `0.3069` mean test
  macro-F1.

Conclusion: current data supports high F1 for the OМСУ selective layer, but not
for every detailed taxonomy axis. Further work should focus on contrastive
pseudo-gold per axis, not longer broad training.
